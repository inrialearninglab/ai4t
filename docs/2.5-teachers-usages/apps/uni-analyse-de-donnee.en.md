**Interview Analysis**

* [NVivo](https://www.qsrinternational.com/nvivo-qualitative-data-analysis-software/home) (with integrated AI)
  Qualitative data analysis tool. AI can analyze text corpora and help identify themes, patterns, or relationships.

---

* [MAXQDA](https://www.maxqda.com/) (with integrated AI)
  Qualitative analysis software using AI to automate data coding, facilitate transcription analysis, and identify relationships between variables.
* Example: A marketing student conducts a qualitative study on consumer behavior and uses MAXQDA to code interviews and detect trends and insights.

---

**Economic Data Analysis**

[DataRobot](https://www.datarobot.com/) (for predictive analysis)
Data analysis platform using AI to automate the processing of large datasets and generate forecasts to better understand trends.

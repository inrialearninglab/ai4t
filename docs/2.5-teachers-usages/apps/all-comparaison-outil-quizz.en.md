This table presents quiz creation tools mentioned by teachers on the AI4T forum. Additional information (e.g., country of origin) was extracted from the official websites of each tool.

| Tool                          | Tool Description                           | Nationality / Country of Origin | Learner Sign-up Required             | Limitations / Comments                                               |
| ----------------------------- | ------------------------------------------ | ------------------------------- | ----------------------------------- | -------------------------------------------------------------------- |
| [Quizizz](https://quizizz.com/) | Formative assessments, detailed reports    | United States                   | **No** (game code sufficient)       | Less flexible for very specific needs                                 |
| [Quiz Wizard](https://www.getquizwizard.com/) | Quizzes generated from YouTube videos     | Ireland                         | **No** (direct link)                 | Strongly depends on suitable videos, mostly English interface        |
| [Kahoot!](https://kahoot.com/) | Gamified, dynamic learning                  | Norway                          | **No** (PIN code)                    | Not ideal for formal assessment                                       |
| [Nolej AI](https://nolej.io/)  | Automatic generation of H5P quizzes        | France                          | **No** (H5P does not require an account) | Questions auto-generated, requires review                              |
| [Quizlet](https://quizlet.com/) | Flashcards and interactive quizzes         | United States                   | **Yes** (account required)           | Many features locked without subscription                             |

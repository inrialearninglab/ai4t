description: Outils IA de création d'images mentionnés dans le forum du Mooc
---

Ce tableau présente les outils de génération d’images mentionnés par des enseignants sur le forum du Mooc <a href="https://www.fun-mooc.fr/fr/cours/intelligence-artificielle-pour-et-par-les-enseignants-ai4t/" target=_blank>"Intelligence artificielle pour et par les enseignants"</a> - consulté le 01/07/2025.  
Les informations complémentaires (Description de l’outil ; Inscription obligatoire ; Limites techniques ; Pays d’origine) ont été extraites des sites officiels de chaque outil.

---

| Outil                                                                         | Description de l’outil                                  | Inscription obligatoire | Limites techniques                               | Nationalité / Pays d’origine |
| ----------------------------------------------------------------------------- | ------------------------------------------------------- | ----------------------- | ------------------------------------------------ |------------------------------|
| [Krea.ai](https://www.krea.ai/)                                               | Génération d’images à partir de descriptions textuelles | Oui                     | Durée et résolution limitées en version gratuite | Etats unis                   |
| [Craiyon](https://www.craiyon.com/en)                                         | Génération d’images open source à partir de texte       | Non                     |                                                  | États‑Unis                   |
| [DALL‑E (OpenAI)](https://openart.ai/home)                                    | Génération d’images avancée basée sur GPT‑4             | Oui                     | Crédits gratuits limités                         | États‑Unis                   |
| [Janus Pro](https://janusai.pro/)                                             | Génération d’images via DeepSeek                        | Non                     |                                                  | Chine                        |
| [Dream Lab (Canva)](https://www.canva.com/fr_fr/help/generate-with-dreamlab/) | Génération d’images intégrée à Canva                    | Oui                     | Accès limité via Canva Pro                       | Canada                       |
| [Ideogram](https://ideogram.ai)                                               | Création d’images avec texte intégré                    | Oui                     | Crédits limités en version gratuite              | États‑Unis                   |
| [Playground](https://playground.com/)                                         | Génération et édition d’images interactives             | Oui                     | Limites sur exports gratuits                     | Etats Unis                   |
| [Leonardo.ai](https://leonardo.ai/)                                           | Génération avancée avec API personnalisable             | Oui                     | Crédits limités                                  | États‑Unis                   |

## Description[^1]

Wooflash is a Belgian platform that allows creating educational resources, tracking student progress, and planning reviews with an adaptive learning system.

## Features[^1]

* Generation of interactive educational modules (including quizzes) adapted to learners’ levels.
* Progress tracking: analyzes student performance and adapts content.
* Detailed statistics: provides precise feedback on students’ strengths and gaps.

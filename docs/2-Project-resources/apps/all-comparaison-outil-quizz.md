---
title: Synthèse d'outils IA de génération de quiz
description: Outils IA de génération de quiz mentionnés dans le forum du Mooc
---

Ce tableau présente les outils de création de quizz par des enseignants sur Le forum du Mooc "[Intelligence artificielle pour et par les enseignants](https://www.fun-mooc.fr/fr/cours/intelligence-artificielle-pour-et-par-les-enseignants-ai4t/)" - consulté le 01/07/2025.
Les informations complémentaires (exemple : nationalités) ont été extraites des sites officiels de chaque outil.

| Outil                         | Description de l’outil                     | Nationalité / Pays d’origine | Inscription apprenant requise          | Limites / Commentaires                                               |
| ----------------------------- | ------------------------------------------ | ---------------------------- | -------------------------------------- | -------------------------------------------------------------------- |
| [Quizizz](https://quizizz.com/) | Évaluations formatives, rapports détaillés | États-Unis                   | **Non** (code de jeu suffisant)        | Moins flexible pour des besoins très spécifiques                     |
| [Quiz Wizard](https://www.getquizwizard.com/) | Quiz générés à partir de vidéos YouTube    | Irlande                      | **Non** (lien direct)                  | Dépend fortement de vidéos appropriées, interface surtout en anglais |
| [Kahoot!](https://kahoot.com/)| Apprentissage ludique, dynamique           | Norvège                      | **Non** (code PIN)                     | Pas idéal pour de l’évaluation formelle                              |
| [Nolej AI](https://nolej.io/) | Production automatique de quiz H5P         | France                       | **Non** (H5P ne demande pas de compte) | Questions générées automatiquement, nécessite relecture              |
| [Quizlet](https://quizlet.com/) | Flashcards et quiz interactifs             | États-Unis                   | **Oui** (compte requis)                | Plusieurs fonctions verrouillées sans abonnement                     |

## La Méthode A C T I F

- **A pour Action** : Précisez ce que vous attendez de l’IA.  
Exemple : générer un résumé, analyser les données.

- **C pour Contexte** : Définissez clairement le contexte dans lequel l’IA doit intervenir.  
Exemple : pour une présentation en classe, pour un projet de recherche.

- **T pour Tâche** : Soyez explicite sur la tâche que l’IA doit accomplir.  
Exemple : rédiger un texte sur l’évolution de la biologie, comparer deux articles scientifiques.

- **I pour Input** : Donnez les informations ou les données dont l’IA a besoin pour effectuer la tâche.  
Exemple : voici les données de l’expérience, voici un extrait d’un texte à résumer.

- **F pour Format** : Spécifiez le format de la réponse attendue.  
Exemple : un tableau, un texte de 300 mots, un graphique comparatif.

## Les Cinq Étapes pour Créer un Bon Prompt (selon la DRANE)

- **Initialiser la conversation** : commencez par définir votre rôle et votre objectif.  
Exemple : je suis professeur de français et je souhaite créer une leçon sur le romantisme.

- **Décrire la tâche** : soyez clair et spécifique sur la tâche demandée.  
Exemple : je souhaite que vous génériez un plan détaillé pour cette leçon sur le romantisme.

- **Ajouter des contraintes** : précisez les limites ou spécifications nécessaires.  
Exemple : le cours s’adresse à des élèves de collège et doit durer 45 minutes.

- **Préciser le format de sortie** : indiquez comment la réponse doit être présentée.  
Exemple : veuillez fournir un plan sous forme de tableau avec des colonnes pour les objectifs, les activités et les évaluations.

- **Affiner les réponses** : ajustez le prompt en apportant des précisions ou en reformulant la demande.  
Exemple : pouvez-vous détailler davantage les activités en classe et suggérer des ressources spécifiques pour chaque section ?

## Exemples Pratiques de Création de Prompts

### Exemple 1 : Créer un résumé d’article scientifique

Action : résumer l’article  
Contexte : pour une présentation lors d’une conférence scientifique  
Tâche : extraire les points principaux de l’article  
Input : l’article complet à résumer  
Format : résumé de 300 mots

Prompt : Je suis chercheur en biologie et je prépare une présentation pour une conférence. Je souhaite que vous résumiez l’article suivant sur les effets du réchauffement climatique sur la biodiversité, en extrayant les points principaux. Le résumé doit comporter environ 300 mots.

### Exemple 2 : Création de quiz personnalisé pour les élèves

Action : créer un quiz  
Contexte : pour un cours de mathématiques au collège  
Tâche : générer 10 questions sur les fractions  
Input : liste des compétences des élèves sur les fractions  
Format : questions à choix multiples avec réponses détaillées

Prompt :  Je suis professeur de mathématiques au collège. Je souhaite créer un quiz sur les fractions pour mes élèves. Génère 10 questions à choix multiples avec des réponses détaillées pour chaque question. Assure-toi d’adapter les questions en fonction du niveau de compétence de mes élèves, qui vont de débutants à intermédiaires.

## Conseils Pratiques

✅ Tester et ajuster les prompts : ne vous contentez pas de la première réponse. Affinez le prompt pour obtenir des résultats plus pertinents.

✅ Utiliser des modèles adaptés : certaines plateformes d’IA offrent des modèles de prompts que vous pouvez utiliser comme point de départ, avant de les personnaliser.

✅ Vérification des résultats : toujours vérifier les résultats fournis par l'IA et ajuster les paramètres ou le format du prompt si nécessaire pour améliorer la qualité des réponses.
